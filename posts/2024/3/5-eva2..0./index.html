Eva2.0

82

小白神秘兮兮地把我与祁时拉到一边，眼睛一眨不眨，极为振奋地目视我们。

「还记得我们之前聊过能不能毁掉 Eva 吗？」

记得，那已经是很久以前的讨论了。

全区的电子化管理程度极高，而 Eva 就是这个世界的超级大脑。

顶层的数据中心收集来自全区的信息，制定出了一套利于自身的社会规则，大到税收条款，小到公共交通。又通过强有力的军务手段，维系着这套法则的运转。

Eva 身上凝聚了科技司所有职工的心血，汇集成了一套庞大又复杂的管理系统。科技司无数顶尖的计算机高手时刻盯紧 Eva，修复出现的任何一个 bug。

如果 Eva 被毁，全区的运行也会受到影响，大多数现行规则都会被按下暂停键，人们的生活也会暂时陷入混乱。

各区任何涉及电子化执行的场域，会全部失效！

A 区 B 区 C 区出入通道的权限判断完全失效，无法判别任何车辆是否有通行资格；全区数据库崩溃，昨天还好好记录的人口、经济各类信息，会瞬时「消失」；各行业交易的价码无矩可依，B 区坐个公交车都无法支付……

简单来说，就是一个电子化交易为主的时代，背后的隐性规则突然被清除，仿佛电子时代瞬间回到工业时代。

「是，确实……但科技司那群人也不是吃干饭的，迟早会修复好。就算没有 Eva，他们再造一个新的管理系统，也只是时间问题。」

小白嘚瑟地抖了抖眉毛，「是，所以我想的不是毁掉它，而是改变它的规则。」

在来军务中心的路上小白已经初步说过他的想法了。他所说的自己在做的那个大项目，绝对伟大的设想。

潜伏到科技司就是为了了解 Eva 的运作，而了解后，小白逐渐在原基础上调整改进，搭建了一套自己心目中更为合适的运转规则。

「A 区长期以来压着 B 区和 C 区，真的是全区资源不够吗？我很好奇这个问题，去科技司以后做了更多详细的计算。」

「计算的结果根本不是那样，全区的资源足够各区以相对公平的水准生存。」

小白所说的这点在前首席的话中得过印证。

以这个世界的资源，并非不足以支撑各区在一个相对良好的水平线生活。但 A 区需要优越感，需要位居他人之上，那种出自本能的贪婪，加剧了 A 区向下吸血的力度。

「明明 A 区只要放弃一些东西，B 区和 C 区就不用再过这种日子，但这群人就是不肯放手。」小白眼中似乎能迸出火光来。

「所以靠人性，是无法做到公平的。没有感情的机器，反而能够做到公平。Eva 本身也没有私欲，只是受到了设计者的影响。」

「那么，如果有一套新的 Eva 呢？它不为 A 区而生，不倾向于任何一方，这样的系统你们能想象吗？」

越说小白越兴奋，眼眸也越来越亮。和任何一个科技宅一样，小白永远奉行的是科技至上的准则。

他相信机器在某种程度堪比神明，他对技术如痴如醉。

在机房里无数偌大的屏幕之间，在低声轰鸣的主机之间，靠着一瓶饮料就能度过一个下午，将０和１化作一个世界的，小白，他是自己勾勒出的世界的神明。

我和祁时屏住呼吸，听着小白接下来的惊世之言。

「我在 Eva 基础上做了一个新模型，首先按照现在的个人所得计算目前的各区差距系数，然后以减少各区差距为目标，计算各种输出项合理的规则设定。」

我跟祁时盯着他，就像两个专注听故事的小朋友。

「说太细了你们也不懂，我就用人话解释好了。」小白耸耸肩，但我们谁都没功夫去嘘他。

「简单说就收入和支出，有出有进。A 区现在之所以维持着这样的水准，因为 B 区和 C 区的成果无条件上贡，自己只能留下很少的部分。」

「这个模型最重要的有几大准则。」

「第一，A 区现有的财富需要定期拿出大比例进行 B 区和 C 区的建设。」

「第二，B 区和 C 区的劳动成果不能再被窃取，而需要以合理的价格流转。那样每个环节只会获得一定的利润，不会最后全给 A 区白嫖。」

「这是核心原则，实际运转上所有的输出项，都会以各区差距系数缩小为目标，进行自动化最优分配。」

「这个模型嘛，我叫它 Eva2.0。」小白骄傲地吐出名字，像介绍自己的伟大发明。

我们也眼睛为之一亮。

长久以来，A 区是靠着说一不二的权力，维系着这套利于自己的全区法则。核心表现上是物质利益的垄断，以及无法随意流动、进行分区跃迁的人口体系。

小白所说的能够解决前者。A 区大规模进行财富转移，而 B 区和 C 区则进行收入增加，两者叠加之后带来各区差距的缩减。

如果这点确实能解决，A 区和 B 区 C 区的云泥之别被缩小，那后者也不攻自破。是否向 A 区流动，已经不再重要。

但，很快又黯淡下来。

「这个设想确实很棒，但还是有一样的问题。我们黑掉了 Eva 后，他们可以想办法修复，或者再构建新的系统。」

设想很美丽，现实很残酷。

「是没错。」小白叹口气有些惆怅，「这是唯一难解的问题。」

「一般怎么处理这种问题……如果他们被威胁了，不得不用呢？比如说上面规定必须这么做之类的，但上面肯定不会这么规定，那么要怎么样才能……」

小白语速飞快地开始描述他的内心戏。

「威胁？」

「他是说，如果不这么做会有其他更坏的后果，那就不得不做了。」祁时淡淡补充道。

我笑了，「会有什么威胁，不做难道 A 区就要爆炸么？……爆炸？」

我一惊，震惊又狐疑地盯着小白，「你认真的？」

小白挤眉弄眼：「没错，不愧是 453 的女人，脑子很灵光啊。我本来是设了一个假变量在那，比如 A 区毁灭之类的。」

「这个模型的运转法则说过了，但它还有另一个重要的规则，就是强制使用规则。」

「能想象吧，如果 Eva 被黑他们有两种选择，要么改动 Eva，要么推翻 Eva，重新构建一套电子化管理系统。但只要不按照我设定的规则运转，或者使用量级环比变动超出异常值，就会触发那个毁灭变量。」

「至于这个毁灭变量嘛，本来只是一个设定，因为我想好好做完最起码保证完整。但现在看，它不就在眼前现成的吗？」

小白语气兴奋，像喝了几两白的。

我先是倒抽一口凉气，目瞪口呆，然后狠狠地拍了把他的肩膀。

「小白，你真他娘的是个天才！」

小白耸肩，「一直都是，不然怎么好意思跟 453 站在一起呢？不对，你好好说话，别他 x 的他 x 的……」

不过我心里想的是，跟 453 站在一起的是我好不，已经没有第三个人的位置了呢。

「噢对了还有一个点，新模型是以降低各区差异为目标的，所以不能少了人均所得这个指标，最后的目标是平均个人所得差距的缩小」

「哎也多亏有你啦。你记得你之前做的 C 区改善计划吗？C 区当时有不少黑户呢，那次做了一次彻底的大检查。这下他们全在数据库里了。」

确实，当时为了改善基本的设施，做过人口调查来确定基本设施的合理密度。C 区本来是电子化程度相对最低的区，经过那次以后，基础数据也全部完整化了。

无数的珠子仿佛串联成线，线又汇聚成面。

过去发生的一切，那些像是巧合又像是随机的事件，在某个瞬间汇沙成塔，在这一刻显得不可思议。

仿佛命运把我们推到这里，仿佛你经历过的一切，都在将你指引前往这个方向。

我们站在这个足以引发惊天动地变化的开关前，随时可以按下。但机会只有一次。

「改变世界的机会就在眼前，要启动么？」

小白面露期待之光地看着我们，我们几乎没有犹豫地回应：「动手吧！」

改变世界不是我们的职责，也难以改变。前首席、老 J，无数的人困在这个利益纠葛的大网里，有的人想要改变无力改变，有的人牵一发而动全身，有的人逐渐被同化失去那些热血的初心，有的人享受既得利益得过且过……

真正有决心与毅力、抱着失去一切的准备去改变世界的人，大概只有真正的自觉者，比如前首席口中那个朋友，以及真正被苦难压迫的人们。

但又何尝容易。

面临的是物质条件差距极大的现实，极度复杂的人心，可能缺乏足够智慧与公正的顶层设计……

而现在，一个另辟蹊径的机会突然出现在眼前。

像一道流星，突兀耀眼。不可思议得像是不真实的，随手就能被戳破的泡影。

更重要的，它转瞬即逝。

「做吧，还能比现在更糟吗？」

反正左一刀又一刀的。前有虎视眈眈、还没有完全被斗争失败的赵冶他们，后有准备突袭闯进 A 区的假老 J，不管怎么说我跟祁时都不得安宁。

还不如趁现在，抓住这个稍纵即逝的机会，就算不一定能成功，但至少也做出了行动。

说干就干，按小白的嘱咐，我们需要帮他进入军务中心的高能量武器控制中心，在那里关联 Eva2.0 与爆破区爆破装置的关系。

因为军务司物资独特，武器控制单独开辟了自己的管理分支，并不直接和 Eva 挂钩。

紧锣密鼓筹划好这些后，回头就看到了卡隆。刚刚我们三小声说了一堆，也不知道他听到了多少。但他并没有说什么。

「将军。我们需要进入高能量武器控制中心，请您暂时疏散下其他人，防止打扰科技司人员的工作。」我凝视他的眼睛，自然地撒谎道。

解决炸弹是否需要进入军务中心的核心，卡隆作为经验老道的将军，应该比谁都明白。

卡隆深褐色的眼睛冷冷盯着我们。军务司内的冷气打得很足，但也不及卡隆的目光犀利。他在掂量着什么，或许不只是思考话语的真实程度。

隐约能感觉到，他也知道我们进去，是要做某项了不得的事。

「将军，全区人的命运，现在就掌握在您的手中。」

他没有直接下令把我们抓起来，说明他内心已经在掂量权衡了。

我继续加码：「将军维护全区的正义，一直令人钦佩。」

「但将军您有没有想过，自己在维护的正义，真的是正义么？」为了少部分人而生的「正义」，还能称得上是正义么？

三双眼睛紧张地盯着卡隆。

能否成功的第一道，就在于卡隆是否放行。

83

卡隆笔直矗立的身躯，即使在任何时刻也如屹立的丰碑，散发着不受动摇的威严。

他冰冷地凝视了我们半晌，声音依然如钢铁般冷峻：「配合科技司人员的一切要求，务必在爆炸前解决！」

有卡隆这句话，相当于为我们放行了。

我们需要两边跑，搞定爆破区仓库假老 J 安置的棘手的炸弹，同时去控制中心。小白借口说需要多个环节，其实解决炸弹并不需要在控制中心。

而后，一位身穿军服的人员在卡隆的指示下为我们带路。

撇散了众人后小白开始小声吐槽：「他竟然没阻止我们，不会是着急上头了吧。」

应该并不是着急所致。卡隆的目光依然绝对理智和清醒，保证在任何情况下，都做出及时果断的决策。

或许，所经历的一切已经开始让他怀疑，自己在捍卫的正义，是否是真正的正义？

心中顿时有种尊敬的情绪油然而生。

卡隆是顶层目前为止，我见过的唯一令人尊敬的人。他有真正的将军风范。

进了控制中心后，小白一顿操作猛如虎，对着这个控制台摸摸，对着那个大屏幕瞅瞅，很快摸清楚了控制中心内的操作系统。

他手指动得飞快，在带来的操作终端上噼里啪啦。

「炸弹已经解决了，它非要时间那就给它时间，设置了一个未来无限远的时间，军务司这帮家伙肯定能近期搞定。」

「至于关联，关联这里所有最强的武器吧，再设定一下位置绝对位移不能超过半径三百米，防止回头他们把这里清理掉。」

要做就要做点大的，小白完美地贯彻了这个准则。

我跟祁时眼巴巴入迷地盯着，除了帮他递递水、扇扇风，倒是也做不上什么。

已经到了后半夜，黎明的晨光很快就要降临。

「就等 Eva2.0 覆盖掉线上的 Eva 了，等着吧。」

最后，小白停下了手里的动作，而我们注视着屏幕里的进度条一点点前移。

就在这时，门外传来了敲门声。

我狐疑地走过去，打开通讯开关对外面说话。控制中心的大门严丝合缝，质地厚实，隔绝了任何外界的动静。

「谁？进程还没有完毕，现在还不能被打扰。」

「姐姐，是我哦。」

从扬声器传来孩童的声音，带着熟悉的清脆和欢快。

路西？他怎么会来？

「姐姐这么久了好不容易才回来，我想第一眼看到姐姐呢。姐姐不联系我，我只能自己过来了。」路西的腔调一如既往的天真，但我们都知道表象下面掩藏着其他东西。

「开门吧。」

我如鲠在喉，犹豫着要不要开。在这个关头路西突然过来，总觉得不太对劲。

「开门吧，如果不开我现在就联系叔叔，告诉他你们在做的好事。」

……还学会威胁了。

我准备开门，祁时皱眉。

「没事，我自己出去就行，你在这里保护小白，千万保证要支撑到发布完。」我低声道。

「啧啧，就是。453 你天天跟乔思待在一起还不够嘛，好不容易有时间来点咱们男人之间的独处。」

门打开一个人通过的缝，然后被再度关闭。

路西依然穿得精致，昂着头看我，像个骄傲的小贵族。

我们找了个不远处的僻静角落，我动了动嘴，一时间不知道该说些什么。

「姐姐变了呢，都不跟我笑眯眯地说话了。」他突然吐出这么一句话，语气依然是平静的。

我咧开一个可能不太好看的笑容，「对不起，最近发生太多事了。」

不过瞬间感受到了某种悲怆，于是立刻又展示出了一个更自然的笑容，「等一切都安稳下来，我陪路西好好玩怎么样？」

路西有什么错呢，他也只是个半大的孩子。对着孩子，永远都要展现乐观的一面。

路西笑了，「姐姐在说什么呢，自己都自身难保了，还想着要安慰我吗？」

我有些困惑而谨慎地看过去，他笑盈盈的，还是天真无害的样子，嘴里说出的话却让人摸不清头脑。

「这里炸就炸了，为什么要来阻止呢？都是一些烂人而已，毁掉也无所谓。」

我看了他两秒，总觉得有不对劲的地方。他说的是「这里」，并不是指那 4 栋被要求人群疏散的建筑，应该说明他也知道军务中心被安装了炸弹。

知道军务司内部的事，还在我回来后第一时间就赶过来。他，应该不是为了看我才特地过来的。

「我们费劲心思才把炸弹安插在这么重要的地方，姐姐你怎么能阻止这一切呢？」

路西的话像是一座已经预想到会爆发的火山，但，还是在眼前爆发了。

所以路西，因为姣好的外形从 C 区被挑中，然后迈入另一个光鲜亮丽的深渊。假老 J 是什么时候跟路西搭上线的？早在 C 区就已经开始了，又或许是后来？

「姐姐知道么，我很久就没有家人了。」

「在 C4 那个恶心的地方，没有一个落单的孩子能活下来。我们只能乞讨、捡东西吃，睡在没人注意的大街上，连个挡雨的棚子都没有。身上每天都是臭的，臭到想把自己揉烂。」

「但他们说我很漂亮，可以过上更好的生活。」

「我跟着他们走了，我想过上更好的生活。那里人很多，有很多长得很漂亮的人，但他们看起来又像是死了。我不知道等待我的是什么，也许是更恐怖的东西。」

「有天，我看到一个男孩被打……有血从下面流出来，好像是肠子流出来了……我不知道，我后悔了，我想着要不要回去乞讨算了。」

他在颤抖，他的声音毫无波澜，语速比平时要快，似乎是想尽快跳过这部分。但又分明能听得出他在颤抖，把自己剖开，回忆最黑暗的往事。

我眼睛湿了。

「后来，有人过来把我带走了。然后我来到了这里，第一次见到这么高的高楼，这么多闪闪发亮的东西。可以吃最好吃的东西，把最漂亮的衣服穿在身上。叔叔说我很好看，让我跟着他做他的亲人，我以为我会过上好生活了。」

「可是，可是……」路西低着头，声音变得极轻，像是已经说不出话来。

「我终于知道那个被打的男孩，为什么肠子会流出来……」

「我觉得我好恶心，每天我都想把自己揉烂……」

「别说了。」

我终于忍不住上去，一把上去将路西抱住，抱着他开始哭。我半俯着身子，头靠在路西的肩膀旁，压抑不住地开始哭。

为什么呢，为什么会发生这些事呢？本不该在任何一个世界发生的事，为什么会让路西经历这些呢？他明明，只是个孩子。

我觉得好难过，胸口好像被掏出了一个大洞。

见证到了路西庞大的悲伤，但根本丝毫没有办法。

他就这么一个人待在黑暗泥泞的深渊里，日复一日重复着痛苦。

路西却更平静了，「所以姐姐你知道为什么了吧，他们全都该死，该死！」

我只是抱着他，忍不住汹涌的泪意，嘴里则一直念着：「对不起，不是你的错……不是你的错……」

我就那样抱着路西哭着，两个身影在偌大的军务中心相互依偎。眼边全是湿润温暖的眼泪的触感，分不清是我的还是路西的。

他哭了么，我不知道，直到最后归于平静。

「所以姐姐，现在还决定阻止这场爆炸吗？」路西仰着头看我，像一朵美丽又脆弱的玫瑰花。

我吸了吸鼻子，还是点了点头。

「我不知道怎么说了，也许真的他们都该死，或许有部分人是不该死的。但，但……」

我说不出话来了。

想说的很多，比如毁灭或许并不是最佳解决问题的手段；比如还有无数人想要建设一个更公正更美好的世界，甚至有人付出了自己一切；比如要怀有希望，通过更多的努力，也许以后就不会再有人经历他经历的事……

但这一刻在路西面前，我什么也说不出来了。

他就在那里，经历了无数的黑暗，那些痛苦实质性地砸在他身上，然后你劝他要追求光明？怎么能啊，怎么能做到呢。

路西有些失落地低下头，微长的睫羽垂下遮住了他的眼神。

然后是长久的沉默。

他只要现在去联系首席，或者假老 J 那边的人，我们的行动都会受到阻碍。首席那边可能会派人把我们先逮捕了再说，假老 J 可能安排最近的人过来把我们除掉。

他又抬起头，只是问了一个问题。

「姐姐，之前，你为什么要说带我走呢？」

我一怔，想起了他在问什么。

在首席家谈判后，在以祁时为目标的猎杀五分钟开始前，路西曾给我们指路。那时候我问他，愿不愿意跟我们走。

「因为，因为你看起来很绝望，好像在求助一样……」

尽管从未开口，尽管被高贵的装扮和矜漠的表情武装，依然能听到微弱的呼救声。

在那深不见底的深渊处，孤独地传达着。

「……」

路西没说话。

那晚，路西最后没有举报我们。他走了。

我回到控制中心，跟祁时、小白一起看进度条逐渐抵达 100%，然后正式到达 100%。

没有礼花，没有掌声，那一瞬间安静无比，甚至都难以想象 Eva2.0 是真的发布成功了。

但确实发布了。A 区瞬时陷入瘫痪，但醒着的人们还没来得及搞清都发生了什么，就被四桩爆炸吸引了注意力。

在凌晨曙光到来之时，那四座建筑轰然倒塌。爆炸的威力很强，几乎将周边几个街区入睡的人都震醒。

附近所有还醒着的人，被疏散找了安全区躲避的人，都看到了建筑倒塌的场面。

直至云霄，见证着财力与物力，即使在夜晚也显得无比恢弘的高楼，坍塌了。

巴别塔坍塌了，从此 A 区不再掌握神迹。
