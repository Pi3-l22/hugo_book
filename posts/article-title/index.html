30分钟入门C语言教程

欢迎来到C语言的世界！在接下来的30分钟里，我们将快速了解C语言的基础知识，并通过一些有趣的代码示例和小游戏项目来巩固这些概念。准备好了吗？让我们开始吧！

C程序结构

每个C程序都由一个或多个函数组成，最基本的结构如下：

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
  • #include <stdio.h>：引入标准输入输出库。
  • int main()：程序的入口点。
  • printf：用于输出文本到控制台。

C基础语法

数据类型

C语言支持多种数据类型，包括：

  • int：整数
  • float：浮点数
  • char：字符

变量与常量

变量是可以改变的值，而常量在定义后不能改变。例如：

int age = 25; // 变量
const float PI = 3.14; // 常量

存储类

C语言有四种存储类：auto、register、static和extern。它们决定了变量的生命周期和可见性。

运算符

C语言支持多种运算符，如算术运算符、关系运算符和逻辑运算符。例如：

int sum = a + b; // 加法
if (a > b) { /* ... */ } // 关系运算

控制结构

判断

使用if语句进行条件判断：

if (age >= 18) {
    printf("成年人\n");
} else {
    printf("未成年人\n");
}

循环

C语言支持for、while和do-while循环。例如，打印1到5的数字：

for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

函数

函数是C语言的基本构建块。定义一个简单的函数：

int add(int a, int b) {
    return a + b;
}

作用域规则

变量的作用域决定了它的可见性。局部变量只能在定义它的函数内访问，而全局变量在整个程序中可见。

数组与字符串

数组

数组是一组相同类型的数据。例如：

int numbers = {1, 2, 3, 4, 5};

字符串

字符串在C中是字符数组，以空字符\0结束。例如：

char name = "Alice";

结构体与共用体

结构体

结构体用于组合不同类型的数据：

struct Person {
    char name;
    int age;
};

共用体

共用体允许在同一内存位置存储不同类型的数据：

union Data {
    int intValue;
    float floatValue;
};

位域与typedef

位域允许在结构体中定义特定数量的位：

struct {
    unsigned int isAlive : 1;
} status;

typedef用于定义新类型名：

typedef unsigned long ulong;

输入与输出

使用scanf和printf进行输入输出：

int num;
printf("输入一个数字: ");
scanf("%d", &num);

文件读写

C语言提供了文件操作的功能：

FILE *file = fopen("data.txt", "r");
if (file) {
    // 读取文件内容
    fclose(file);
}

错误处理与递归

错误处理

使用errno和perror进行错误处理。

递归

递归是函数调用自身的过程。例如，计算阶乘：

int factorial(int n) {
    return (n == 1) ? 1 : n * factorial(n - 1);
}

可变参数与内存管理

使用stdarg.h处理可变参数：

#include <stdarg.h>

void printNumbers(int count, ...) {
    va_list args;
    va_start(args, count);
    for (int i = 0; i < count; i++) {
        printf("%d ", va_arg(args, int));
    }
    va_end(args);
}

动态内存管理使用malloc和free：

int *arr = (int *)malloc(5 * sizeof(int));
free(arr);

命令行参数

通过main函数的参数获取命令行输入：

int main(int argc, char *argv[]) {
    printf("参数个数: %d\n", argc);
}

安全函数与排序算法

使用安全函数如snprintf避免缓冲区溢出。简单的排序算法示例（冒泡排序）：

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++)
        for (int j = 0; j < n-i-1; j++)
            if (arr[j] > arr[j+1]) {
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
}

小游戏项目示例

为了巩固所学知识，我们可以尝试实现一个简单的“猜数字”游戏：

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int number, guess;
    srand(time(0));
    number = rand() % 100 + 1; // 生成1到100之间的随机数

    printf("猜一个1到100之间的数字：\n");
    do {
        scanf("%d", &guess);
        if (guess > number) {
            printf("太高了！\n");
        } else if (guess < number) {
            printf("太低了！\n");
        } else {
            printf("恭喜你，猜对了！\n");
        }
    } while (guess != number);

    return 0;
}

通过这个小游戏，你可以实践输入输出、条件判断和循环等基本概念。
